# 📊 E-Commerce Sales Data Analysis

An end-to-end **Exploratory Data Analysis (EDA)** project on an E-Commerce (Superstore) dataset using **Python, Pandas, Matplotlib, and Seaborn**. This project focuses on transforming raw business data into meaningful insights through data cleaning, feature engineering, and visual analytics.

---

## 📌 Project Overview

Businesses generate large volumes of sales data every day. This project analyzes an E-Commerce sales dataset to uncover trends in sales performance, customer behavior, product performance, and shipping efficiency.

The workflow follows a real-world data analytics pipeline:

- Data Loading
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Business Insights
- Data Visualization

---

## 🎯 Objectives

- Clean and preprocess raw sales data.
- Analyze sales performance across regions and categories.
- Identify top-performing customers and products.
- Study customer purchasing behavior.
- Evaluate shipping performance and delivery delays.
- Generate business insights to support decision-making.

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/SwapnilNarwade1203/E-Commerce-Data-Analysis.git
```

### Move into project

```bash
cd E-Commerce-Data-Analysis
```

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open notebooks in the following order:

1. DataLoading.ipynb
2. cleaningData.ipynb
3. Sales_Analysis.ipynb
4. Custmore_Analysis.ipynb
5. Shipping_Analysis.ipynb
